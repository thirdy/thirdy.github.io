Blackmarket
The exiles ultimate way to find the best items in wraeclast.

Blackmarket is free and fully open source, see: github page
for the source code. This means this software will always be free, and anyone can take the source code and modify it for improvements.
Read more about open source here.


Download:
Download the latest release here.

How to run:
Double click the jar. Make sure you have java installed, java 7 recommended.

Screenshot


Example search:


1. boots
2. 2h
3. ring 50life
4. amulet 60res
5. armour 90life 50res

Note:


When you first run blackmarket, a configuration file is created in your user home directory called 'blackmarket.language'. e.g.:
C:\Users\olp\.blackmarket\blackmarket.language

This config file can be extended to allow more search terms to be understood by blackmarket.

Hope you find Blackmarket useful. A lot of work still needs to be done. But it's now usable.
